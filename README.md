# README

[Inspired by Decade of Dotfiles](https://evanhahn.com/a-decade-of-dotfiles/)

```sh
git clone https://github.com/davehoch/dotfiles.git ~/.config 
```

Then cd into your home directory, and run the install script.
.config/install

## path

If you are looking to modify the path, try to do it in `/etc/paths` first.
